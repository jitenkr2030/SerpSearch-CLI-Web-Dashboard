# SerpApi Hindi Tutorials - Installation Guide

## Quick Setup

### 1. Prerequisites
- Python 3.8+ OR Node.js 18+
- SerpApi Account (free)
- Git

### 2. Get API Key
```bash
# Visit https://serpapi.com/
# Sign up for free account
# Copy your API key
```

### 3. Environment Setup
```bash
# Set environment variable
export SERPAPI_KEY=your_api_key_here

# Or create .env file
echo "SERPAPI_KEY=your_api_key_here" > .env
```

### 4. Choose Your Path

#### Python Path:
```bash
cd python
pip install google-search-results
python basic_search.py "कृत्रिम बुद्धिमत्ता"
```

#### Node.js Path:
```bash
cd nodejs
npm install
node src/search.js "मशीन लर्निंग"
```

#### CLI Path:
```bash
cd cli-tool
python serpsearch.py "आर्टिफिशियल इंटेलिजेंस"
```
